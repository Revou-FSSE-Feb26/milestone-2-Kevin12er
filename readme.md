# Project Documentation

## Overview of the Project

This project is a collection of interactive web-based games. Each game is designed to be engaging and fun, providing users with a variety of challenges and entertainment. The project is structured to allow easy navigation between games and is built with a focus on simplicity and user experience.

## Features Implemented
- **Game Collection**: Includes multiple games such as:
  - Typing Game
  - Memory Card Game
  - Rock-Paper-Scissors
  - Number Guessing Game
  - Whack-a-Mole
- **Interactive Gameplay**: Each game is interactive and provides immediate feedback to the user.
- **Responsive Design**: The project is designed to work seamlessly across different devices and screen sizes.
- **Organized Structure**: The project files are well-organized for easy maintenance and scalability.

## Technologies Used
- **HTML**: For structuring the web pages.
- **CSS**: For styling and enhancing the visual appeal.
- **JavaScript**: For implementing game logic and interactivity.

## Route Folder
- **Assets/**: Contains additional assets used in the project.
- **Images/**: Stores images used across the project.
- **Javascript/**: Contains JavaScript files for each game:
  - game-ketik.js
  - Memory-card.js
  - rock-paper.js
  - tebak-angka.js
  - whack-a-mole.js

- **pageGame/**: Contains HTML files for each game:
  - game-ketik.html
  - Memory-card.html
  - pageGame.html
  - rock-papper-scissors.html
  - tebak-angka.html
  - whack-a-mole.html
